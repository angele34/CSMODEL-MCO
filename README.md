# Setup Instructions

## Prerequisites

Before running the notebook, make sure you have:

- Python 3.10 or later
- Jupyter Notebook or Visual Studio Code with the Jupyter extension
- `pip` installed


**1. Clone the repository**

```bash
git clone https://github.com/angele34/CSMODEL-MCO.git

```

**2. Cd into project directory**

```bash
cd CSMODEL-MCO
```

**3. Create a Virtual Environment**

```bash
python -m venv .venv
.venv\Scripts\activate
```

**4. Install Dependencies**

```bash
pip install -r requirements.txt
```

## Contributing

Before starting any work:

1. Pull the latest changes from the repository.
2. Check the **Issues** tab and assign yourself an open issue before working on it.
3. Reference the issue number in your commit message.

#### Commit Message Format

Use the following format when completing an issue:

```text
<short description> (#<issue-number>)
```

Examples:

```text
Implement Statistical Inference (#2)
Improve notebook formatting (#3)
```
